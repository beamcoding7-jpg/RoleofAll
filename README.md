# ใครทำอะไร — RIASEC Career Site Team Task Board

เว็บไซต์สำหรับบริหารจัดการและติดตามความคืบหน้าการทำงานรายบุคคลและภาพรวมของทีมงานในการพัฒนาโครงการ **RIASEC Career Site** (โครงงานระบบแนะนำอาชีพตามบุคลิกภาพ RIASEC) ตลอดระยะเวลา 6 สัปดาห์

* **ลิงก์เข้าใช้งานเว็บไซต์:** [https://beamcoding7-jpg.github.io/RoleofAll/](https://beamcoding7-jpg.github.io/RoleofAll/)

---

## 📋 เกี่ยวกับเว็บไซต์

เว็บไซต์นี้ถูกพัฒนาขึ้นเป็น Single-Page Web Application เพื่อใช้เป็นบอร์ดติดตามงานภายในทีมที่มีสมาชิกทั้งหมด 5 คน ช่วยตอบคำถามว่า **"ใครทำอะไร"** ในแต่ละช่วงเวลาของโครงงาน ตั้งแต่ขั้นตอนเริ่มต้นค้นคว้าข้อมูลวิจัยไปจนถึงสัปดาห์นำเสนอผลงานสุดท้าย 

ระบบออกแบบมาให้ใช้งานง่าย มีความยืดหยุ่นสูง และอำนวยความสะดวกในระหว่างการประชุมทีมด้วยฟีเจอร์การปฏิสัมพันธ์ต่างๆ

---

## ✨ ฟีเจอร์หลัก (Key Features)

1. **มุมมองรายคน (Individual View):**
   - ช่วยแสดงรายละเอียดหน้าที่หลัก คำอธิบายบทบาท สิ่งที่ต้องเรียนรู้เพิ่มเติม และเป้าหมายส่งมอบงานสุดท้ายของสมาชิกแต่ละคน
   - แสดงรายการงานแยกตามรายสัปดาห์ (สัปดาห์ที่ 1 - 6) พร้อมกล่องเช็คลิสต์ที่กดติ๊กสะสมคะแนนความคืบหน้าได้แบบ Real-time
   - สามารถแก้ไขชื่อสมาชิกบนหน้าเว็บได้ทันที (โดยการกดคลิกหรือกดค้างที่ชื่อเพื่อสลับเป็นโหมด Contenteditable) เพื่อปรับเปลี่ยนให้เข้ากับชื่อจริงของคนในทีม
   - มีระบบคำนวณและแสดงแถบระดับความคืบหน้า (Progress Bar) สำหรับแต่ละคนโดยเฉพาะ

2. **มุมมองรวมทั้งทีม (Team Overview):**
   - แสดงตารางภาพรวม (Grid View) สรุปงานของสมาชิกทั้ง 5 คนในสัปดาห์นั้นๆ ร่วมกัน เพื่อให้เห็นการทำงานขนานกันของแต่ละฝ่ายได้ชัดเจน

3. **กราฟความเชื่อมโยงมิติบุคลิกภาพ (Interactive RIASEC Radar Hexagon):**
   - ด้านบนมี SVG กราฟรูปหกเหลี่ยมเชื่อมโยงตามหลักการวัดบุคลิกภาพ RIASEC
   - เมื่อกดเลือกสมาชิกแต่ละคน กราฟจะแสดงเส้นเชื่อมโยงและไฮไลต์กลุ่มบุคลิกภาพของสมาชิกคนนั้นโดยอัตโนมัติ (เช่น **I - C**, **A - I**, **E - S**)

---

## 👥 โครงสร้างบทบาทของทีม (Team Structure)

สมาชิกทั้ง 5 คนจะได้รับมอบหมายหน้าที่สอดคล้องตามทักษะและกลุ่มบุคลิกภาพ RIASEC ดังนี้:

| ลำดับสมาชิก | บทบาทหลัก (Role) | รหัส RIASEC | สิ่งที่ต้องส่งมอบสุดท้าย (Deliverables) |
| :--- | :--- | :---: | :--- |
| **สมาชิกคนที่ 1** | **Frontend & UI/UX** | **A / I** | หน้าเว็บทั้งหมดที่ใช้งานได้จริง ตอบสนองทุกขนาดหน้าจอ (Responsive Web Design) |
| **สมาชิกคนที่ 2** | **Logic, Data & Deploy** | **I / C** | ระบบคำนวณคะแนน RIASEC, ฐานข้อมูล และการ Deploy เว็บไซต์ขึ้นระบบออนไลน์ (GitHub Pages / Vercel) |
| **สมาชิกคนที่ 3** | **Content & Career Research** | **I / S** | เอกสารข้อมูลอาชีพ 30+ อาชีพ และสายการเรียน 20+ สายเรียนที่จัดประเภทตรงตามรหัส Holland Code |
| **สมาชิกคนที่ 4** | **Documentation & Report** | **C / S** | รายงานเล่มโครงงานฉบับสมบูรณ์ (บทที่ 1 ถึงบทที่ 5) ตามหลักวิชาการ |
| **สมาชิกคนที่ 5** | **Testing, Survey & Design** | **E / S** | ผลสรุปแบบสอบถามความพึงพอใจจากกลุ่มทดลอง 30+ คน และสื่อสไลด์/โปสเตอร์นำเสนอผลงาน |

## 🛠️ เทคโนโลยีที่ใช้พัฒนา (Technologies Used)

* **โครงสร้างหลัก (Structure):** HTML5 (Semantic Elements)
* **การตกแต่ง (Styling):** Vanilla CSS3 (Custom CSS Properties/Variables, CSS Grid, Flexbox, Keyframes Animation, Media Queries สำหรับ Responsive Design)
* **ตรรกะระบบ (Logic):** Vanilla JavaScript (การจัดการ DOM, การเชื่อมโยง Interactive SVG, การจัดเก็บ State การติ๊กงานและความก้าวหน้าเบื้องต้น)
* **ฟอนต์ใช้งาน (Typography):** Google Fonts (`Kanit`, `Sarabun`, `Mali`)
